FaceBook Image Fix solves the problem with the wrong images preview linked on FaceBook. Easy configuration! Seo Plugin support! YouTube thumb support!

== Description ==

FaceBook Image Fix solves problem with FaceBook LikeIt and Share actions. Sometimes when you share/like one of yours pages can happen that images viewed in FaceBook are wrong. This plugin solves the problem by inserting FaceBook Open Graph or XFBML Tag. Easy configuration! Seo Plugin support!

== Features ==

* Support YouTube Thumbs and WordPress Video Plugin shortcode
* You can chose either XFBML Tag or Open Graph Tag which actually creates a FaceBook fan page for each click on "Like it"
* Support Seo Ultimate and All In One Seo Pack meta data
* Retrieve first 300 chars from content and use it in the description tag.
* Adds FaceBook Open Graph Meta Tag in Homepage, Posts and Pages automatically.
* If posts/pages don't have images it will be replaced by your logo

== Installation ==

1. Upload `facebook-image-fix` folder into the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in the Admin WordPress Panel
3. Go to Settings > FaceBook Image Fix to configure your logo
4. Finish! Check is your link is correct into the FaceBook Linter <https://developers.facebook.com/tools/debug>
